Этот компонент нужен, чтобы показывать важные числа — на графиках, схемах, картах или сами по себе.
Например, количество филиалов или процент прибыли.

Число может быть очень заметным, к нему можно добавить комментарии.

## Содержимое

Главный компонент этого элемента — число. К числу можно добавить [заголовок](#заголовок), [единицы измерения](#единицы-измерения), [иконку](#иконка) и [показать, как оно изменилось](#показатель-изменений).

```tsx
<Stats
  value={2170}
  title="Молний за год"
  iconTitle={<svg />}
  unit="разрядов"
  rate="20%"
  status="success"
  layout="default"
  size="s"
/>
```

### Число

Главный элемент этого компонента — число, которое нужно выделить. Укажите его в свойстве `value`.

```tsx
<Stats value={2170} />
```

<StatsExampleContentValue />

### Значение по умолчанию

Если число не задано (`value=null`), показывается значение по умолчанию. Его можно добавить в свойстве `placeholder`.
Здесь может быть что угодно: число, слово или специальный символ. Если ничего не укажете, поставим прочерк `—`.

```tsx
<Stats value={null} title="В этом году я видел" unit="молний" />
```

<StatsExamplePlaceholder />

### Заголовок

Это надпись над числом, задается в свойстве `title`.

Рядом с заголовком можно показать иконку, укажите ее в свойстве `iconTitle`.

```tsx
<Stats
  value={2170}
  title="Скорость молнии"
  unit="км/с"
  iconTitle={IconLightningBolt}
/>
```

<StatsExampleContentIcon />

### Единицы

Это подпись под числом, обычно тут указывают единицы измерения — км/ч, сутки, баррели или ведерки.
Задается в свойстве `units`.

### Показатель изменений

Это способ показать, как изменилось число за какое-то время (время нужно отображать отдельно).
Задается в свойстве `rate`, появляется справа или внизу от числа (это зависит от [формата](#формат)).

Здесь может быть что угодно, но чаще всё-таки число или процент.

Слева от показателя может быть иконка, её можно добавить в свойстве `iconArrowRate`. Варианты:

- `'up'` — число поменялось в большую сторону (стрелка вверх),
- `'down'` — число поменялось в меньшую сторону (стрелка вниз),
- `'auto'` — иконка подставляется автоматически и зависит от первого числа, которое задано в свойстве `rate`. Если первое число положительное (без знака или со знаком `+`), подставится иконка `'up'`. Если перед числом знак `-` — иконка `'down'`.

> Если иконка задана, и при этом перед самым первым числом в строке `rate` есть знак `+` или `-`, этот знак удаляется.
> Его роль выполняет иконка.

| `rate="+20%" iconArrowRate="up"` | `rate="-20%" iconArrowRate="down"` |
| -------------------------------- | ---------------------------------- |
| <StatsExampleRateUp />           | <StatsExampleRateDown />           |

| `rate="+20%" iconArrowRate="auto"` | `rate="-20%" iconArrowRate="auto"` |
| ---------------------------------- | ---------------------------------- |
| <StatsExampleRateAutoPositive />   | <StatsExampleRateAutoNegative />   |

## Статус

Статус устанавливает цвет показателя изменений. Если показатель изменений не указан, в этот цвет перекрашивается основное число.

За статус отвечает свойство `'status'`. Варианты: `'success'`, `'warning'`, `'error'`, `'system'`, по умолчанию — `'system'`.

| `status="success"`      | `status="warning"`      | `status="error"`      | `status="system"`      |
| ----------------------- | ----------------------- | --------------------- | ---------------------- |
| <StatsExampleSuccess /> | <StatsExampleWarning /> | <StatsExampleError /> | <StatsExampleSystem /> |

## Расположение элементов

За расположение элементов отвечает свойство `layout`. Варианты: `'default'` и `'reversed'`.

По умолчанию показатель изменений находится справа от основного числа, а единицы — под ним (`layout="default"`).

Вы можете поменять их местами (`layout="reversed"`).

| `layout="default"`            | `layout="reversed"`            |
| ----------------------------- | ------------------------------ |
| <StatsExampleLayoutDefault /> | <StatsExampleLayoutReversed /> |

## Свойства

| Свойство                                  | Тип                                       | По умолчанию                            | Описание                                                                                          |
| ----------------------------------------- | ----------------------------------------- | --------------------------------------- | ------------------------------------------------------------------------------------------------- |
| [`value`](#число)                         | `number` \| `null`                        | -                                       | Число, которое нужно показать                                                                     |
| [`placeholder?`](#значение-по-умолчанию)  | `string`                                  | `'—'`                                   | Значение, которое отображается вместо числа, если число не указано (в `value` передан `null`)     |
| [`title?`](#заголовок)                    | `string`                                  | -                                       | Надпись над числом, заголовок — поясняет, что это                                                 |
| [`iconTitle?`](#заголовок)                | `ReactNode`                               | -                                       | Иконка рядом с заголовком, отображается всегда у первой строки                                    |
| [`unit?`](#единицы)                       | `string`                                  | -                                       | Поясняет, в каких единицах измеряется число, меняет положение в зависимости от `layout`           |
| [`rate?`](#показатель-изменений)          | `string`                                  | -                                       | Показатель изменений, меняет положение в зависимости от `layout`                                  |
| [`iconArrowRate?`](#показатель-изменений) | `'up', 'down', 'auto'`                    | -                                       | Иконка для показателя изменений (стрелка вверх или вниз)                                          |
| [`status?`](#статус)                      | `'success', 'warning', 'error', 'system'` | `'system'`                              | Статус, влияет на цвет показателя изменений, если не указан — на основное число                   |
| [`layout?`](#расположение-элементов)      | `'default', 'reversed'`                   | `'default'`                             | Расположение элементов в компоненте                                                               |
| [`size?`](#размер)                        | `'2xs', 'xs', 's', 'm', 'l'`              | `'m'`                                   | Размер компонента                                                                                 |
| `formatValue?`                            | `(value: number) => string`               | `(value: string` \| `number) => string` | Метод форматирования `value`, по умолчанию добавляет отбивки и заменяет точки в дробях на запятые |
| `formatRate?`                             | `(value: string) => string`               | `(value: string` \| `number) => string` | Метод форматирования `rate`, по умолчанию добавляет отбивки и заменяет точки в дробях на запятые  |
