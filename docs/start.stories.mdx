# Stats | Дизайн-система Consta

Компонент для отображения чисел — с заголовком, единицами измерения, иконками и другими настройками. Входит в [дизайн-систему Consta](http://uikit.consta.design/).

[Посмотреть на GitHub](https://github.com/consta-design-system/stats)

[Посмотреть документацию](?path=/docs/components-stats--interactive)

#### Установите пакет и зависимости

`yarn add @consta/uikit @consta/stats` или `npm install @consta/uikit @consta/stats`

#### Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты `@consta/stats` были вложены в компонент `Theme`.

[Как подключить тему](#http://uikit.consta.design/?path=/docs/thematization-how-to-connect-theme--page)
